ⓘ About this Explorer

What is ARGIRA — ARGIRA is a research and accessibility initiative exploring new ways to represent information through technology, including sonification approaches designed to create richer experiences for people with visual disabilities.

What is RMA — Residual Mapping Auditor, the method used to evaluate and validate relationships between variables — distinct from a simple correlation.

What is this Explorer — A tool to browse the dependency network between visual input features and acoustic output parameters, and to see which relationships are verified, exploratory, or executed.

What you can do here — Browse the verified dependency network, inspect the evidence behind each relationship, explore generated hypotheses in Research Mode, and run RMA analysis on your own CSV in the Analysis tab.

What a result does not mean — An exploratory correlation is not a confirmed finding; a Research Mode hypothesis has not been validated; "Executed" means the analysis was run, not that it is verified as general knowledge.

🔗 Correlation Network ⚠ Exploratory (dataset correlations)
Deliberate methodological separation from the Dependency Network above: this is NOT an RMA audit. These are simple Pearson correlations between the chosen target (above, in Target (Y)) and the rest of the numeric columns in the loaded CSV — no control of variables, no cross-validation, no claim of causality. Useful for quick exploration, not as confirmatory evidence. All numeric columns are always shown as nodes (CSV column order); those that don't clear the threshold appear dimmed and without a connection to the target, instead of disappearing from the graph.
Sensitivity values from ±5% perturbation, N=123/227 (104 rows excluded — tempo_space saturated at the 15-BPM cap, making spatial_variance reconstruction from persisted corpus fields unreliable for those rows). roughness and value_mean perturbed as a negative control — both measured 0.0 sensitivity on every output, confirming Layer 1's "not consumed" finding. zones→tempo_bpm/stereo_pan weights are nominal placeholders (not re-derived from this session's perturbation run) — see candidate4_global_representation.md for the source-verified dependency table and full sensitivity matrix.
